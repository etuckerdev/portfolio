SMELS

AI-Powered Error Analysis for Developers

A privacy-first CLI tool that intelligently analyzes programming errors across multiple languages using local AI models, providing instant contextual fixes without compromising sensitive code.

The Challenge

Developers waste countless hours deciphering cryptic error messages, searching through documentation, and debugging across unfamiliar programming environments. This fragmented workflow breaks concentration and significantly impacts productivity, especially when working with complex multi-language codebases.

The Solution

SMELS transforms error debugging into an intelligent, streamlined process. Built with Rust for performance and reliability, it combines sophisticated language parsing with local AI analysis to provide instant, contextual solutions while maintaining complete privacy.

Multi-Language Intelligence

Native parsing for Rust, JavaScript, Python, Java, and more, with smart pattern recognition for accurate error classification.

Privacy-First AI

Local processing via Ollama ensures sensitive code never leaves your machine while leveraging cutting-edge AI capabilities.

Instant Performance

Intelligent caching and optimized Rust implementation deliver sub-second analysis for improved development flow.

Web Interface

Modern web UI with real-time analysis, copy-to-clipboard functionality, and responsive design for enhanced usability.

Technical Architecture

SMELS demonstrates advanced software engineering principles through its modular, extensible architecture. The system employs trait-based polymorphism for parser plugins, implements sophisticated caching strategies, and showcases modern Rust idioms for memory safety and performance.

  • Language Detection: Advanced pattern matching algorithms identify error context across programming languages and frameworks
  • Extensible Design: Plugin architecture enables seamless addition of new languages and error pattern recognition
  • Performance Optimization: Multi-layered caching with TTL management and efficient string processing for minimal latency
  • Graceful Degradation: Intelligent fallback to deterministic analysis when AI services are unavailable

Impact & Innovation
